# TradingStudio Dashboard

TradingStudio is a professional automated trading platform designed for IQ Option. It features elite binary automation, algorithmic signals, and a high-end glassmorphism UI.

## Features
- **AI Execution Engine**: Real-time trade execution based on high-accuracy signals.
- **Premium UI**: Modern chatbot-inspired landing page and dashboard with Framer Motion animations.
- **Security**: Robust session management and military-grade encryption for credentials.
- **Referral Integration**: Integrated referral links for new community members.
- **Live Diagnostics**: Real-time system telemetry and signal analysis stream.

## Getting Started

1. **Install Dependencies**:
```bash
npm install
```

2. **Configure Environment**:
Create a `.env.local` file with your backend URL:
```env
NEXT_PUBLIC_API_URL=https://tradingstudio-backend-production.up.railway.app
```

3. **Run Locally**:
```bash
npm run dev
```

## Deployment
This project is configured for automatic deployment via Netlify. Every push to the `main` branch triggers a new production build.
